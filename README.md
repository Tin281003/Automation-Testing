Hướng dẫn chung:
1. Sử dụng Katalon Recoder để tạo test case test suite sau đó xuất file dưới dạng Katalon Studio 
2. Tải file về và chọn thư mục để giải nén
3. Sử dụng Katalon Studio để mở thư mục đã giải nén file (Chọn File -> Open Project -> chọn thư mục đã giải nén file)
4. Đợi build và chạy code
* Lưu ý nên import lại code để tránh lỗi liên quan đến selenium
